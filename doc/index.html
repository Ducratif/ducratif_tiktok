🚀 Bridge TikTok → FiveM by Ducratif

📌 Prérequis

  • Node.js >= 18
  • npm install

⚙️ Configuration Node

Copiez .env.example en .env puis configurez vos identifiants.

🔥 Commandes TikTok

!confetti → Confettis

!boost → Boost à pied

!boostv → Boost véhicule

!heal → Heal zone

!healme → Heal le streamer

!revive → Revive le streamer

!eject → Eject du véhicule
